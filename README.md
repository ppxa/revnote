# Revnote

中学受験（小4〜6）向けの「リベンジノート」デジタル版。間違えた問題を登録し、忘却曲線（SM-2）に基づく復習で定着を図るアプリ。

- **対象**: 偏差値50帯・塾なし〜中小塾
- **技術**: 単一HTML（Vanilla JS）、localStorage、GitHub Pages
- **現在バージョン**: v5.5
