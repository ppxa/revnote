# revnote v4.0 ロードマップ: 40点→80点

## コンセプト再定義
**旧:** スペースドリピティション問題集アプリ
**新:** 「間違えた問題を最短で定着させる」デジタルリベンジノート + 親の安心ダッシュボード

---

## Phase 1: コア機能修正（今回実装）⚡
> リベンジノートの本質に立ち返る

### 1-1. 間違い登録フローの刷新
- [x] 「📸 間違い登録」に名称変更（「登録」→「間違い登録」）
- [x] クイックモード強化: 写真撮る → 教科選ぶ → 「なぜ間違えた？」選択 → 保存
- [x] 間違い理由テンプレ: ケアレスミス / 知識不足 / 時間切れ / 解法わからず / 問題の読み違い
- [x] 出典フィールド強化: テスト名（組分け/マンスリー/カリテ/宿題/過去問）+ 日付

### 1-2. 解き直し記録の充実
- [x] 問題ごとの手応え記録: ×(全然)→△(惜しい)→○(できた)→◎(完璧) の4段階表示
- [x] 解答時間の問題別記録（タイマーと連動）
- [x] 前回の描画を保存・復元（スクラッチパッドのストロークをproblem.drawingに保存）

### 1-3. 親子モード
- [x] DATA.settings.mode = 'child' | 'parent' 切り替えUI
- [x] 子どもモード: 今まで通りの学習画面（不要な情報を隠してシンプルに）
- [x] 親モード: 学習ログダッシュボード中心
  - 今日の学習: いつ何をやって何分かかって結果はどうだったか（タイムライン表示）
  - 弱点分析: 単元×正答率のヒートマップ
  - 週間レポート: 学習時間推移・正答率推移

### 1-4. 弱点ダッシュボード
- [x] 単元×正答率の一覧（赤/黄/緑のヒートマップ）
- [x] 「3回以上間違えている問題」リスト
- [x] 最近の間違いパターン分析（ケアレスミス率 / 知識不足率）

### 2-3. 描画の進化
- 問題別の描画履歴（前回の解き方を見返せる）
- 画像オーバーレイ描画の保存
